Student Information Center

Overview

The Student Information Center is a web application designed to help university students manage academic records, personal details, and campus announcements from one centralized platform.

Features

  • User authentication and role-based access
  • Academic record management
  • Admin dashboard for content and user control
  • Responsive design with Bootstrap
  • Built with Laravel (PHP), JavaScript, MySQL

Tech Stack

Laravel, Bootstrap, JavaScript, MySQL

My Contributions

I led the frontend development using Bootstrap and wrote RESTful routes in Laravel. I also worked on database structure and implemented user authentication.
